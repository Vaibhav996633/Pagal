# Magical Birthday Surprise 💖

This is a premium, highly interactive React website designed for a magical birthday experience.

## ✨ Features
- **Secure Login**: A glowing, dark-themed login screen requiring a "magic word".
- **Dynamic Intro**: A cinematic typing animation with floating particles.
- **3D Rotating Gallery**: A performant 3D image carousel with reflections and hover-pause capability.
- **Heart Particles**: A continuous background effect of floating hearts.
- **Floating Polaroids**: Randomly floating photos in the background for extra depth.
- **Emotional Slideshow**: A fullscreen journey through memories with touching message overlays.
- **Music Integration**: Autoplay background music with a custom toggle control.
- **Grand Finale**: A heart-burst and confetti celebration with a final message.

## 🔑 Access Details
- **Magic Word**: `forever`
- **Default Recipient Name**: `Beautiful` (Configurable in `src/App.jsx`)

## 🛠️ Assets
- **Photos**: Placed in `src/assets/photos/`. I have generated 6 cinematic placeholders for you.
- **Music**: Place your music file as `src/assets/music/song.mp3`. (I've created a placeholder README in that folder).

## 🚀 How to Run
1. Navigate to the project directory.
2. Run `npm install` (already completed).
3. Run `npm run dev` to start the magical experience.
4. Open the provided local URL in your browser.

## 📂 Folder Structure
- `src/components/`: Modular UI components.
- `src/assets/`: Photos and music.
- `src/App.jsx`: Main logic and flow management.
- `src/index.css`: Custom animations and global styles.

Enjoy the magical celebration! ✨
